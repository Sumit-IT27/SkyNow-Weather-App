# SkyNow - Weather Forecast Web Application

## Abstract
SkyNow is a web-based weather forecasting application that provides real-time weather information using weather APIs. The system displays temperature, humidity, wind speed, pressure, and weather conditions in a clean and responsive interface.

## Features
- Real-time weather updates
- City search functionality
- Responsive design
- Fast performance
- Mobile-friendly interface

## Technologies Used
- HTML
- CSS
- JavaScript
- Weather API
